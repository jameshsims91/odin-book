  - psql (PostgreSQL) 14.23 (Ubuntu 14.23-0ubuntu0.22.04.1)

* Database initialization
  - Create and initialize the database: bin/rails db:setup
    - if db:setup fails please run the steps below:
      - run bin/rails db:create
      - run bin/rails db:migrate 
      - run bin/rails db:seed 
  
  - If you wish to wipe the database entirely and start fresh with seeds,
  please run(only if you know what you are doing and absolutely need
  to wipe the db):
    - bin/rails db:reset
